# 3D Body Map

A web-based application for visualizing and logging pain or symptoms on a 3D human body model.

## Overview

This application allows users to:
- Select different body types (male/female)
- Draw on the 3D model to indicate areas of pain or symptoms
- Complete questionnaires about each pain/symptom
- View a summary of all logged pain points

## Features

- Interactive 3D model visualization
- Drawing directly on the 3D model surface
- Multi-step workflow for pain/symptom logging
- Questionnaire for detailed symptom information
- Summary view of all logged symptoms

## Technologies

- Three.js for 3D rendering
- SurveyKO for questionnaire functionality
- JavaScript modules for code organization
- CSS for responsive design

## Getting Started

1. Clone this repository
2. Open `index.html` in a modern web browser
3. No build process required - runs directly in browser

## Usage

1. Select your body type
2. Add a new pain/symptom by clicking the button
3. Draw on the model to indicate the affected area
4. Complete the questionnaire with details about your symptoms
5. View the summary of all logged symptoms
